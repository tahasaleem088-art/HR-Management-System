# Java DBMS Project

A Java-based Database Management System (DBMS) application developed using **Java Swing** for the graphical user interface and **JDBC** for database connectivity. The application provides an intuitive interface for managing records through Create, Read, Update, and Delete (CRUD) operations while demonstrating object-oriented programming principles and efficient database management.

---

## Features

* User-friendly graphical interface (Java Swing)
* Create, Read, Update, and Delete (CRUD) operations
* Secure database connectivity using JDBC
* Input validation for improved data integrity
* Object-Oriented Programming (OOP) design
* Efficient record management and retrieval

---

## Technologies Used

* Java
* Java Swing
* JDBC (Java Database Connectivity)
* SQL Database
* Object-Oriented Programming (OOP)

## Getting Started

### Prerequisites

* Java JDK 8 or later
* NetBeans IDE (recommended)
* SQL Database (MySQL/SQL Server or the database used in this project)
* JDBC Driver

### Installation

1. Clone the repository.

```bash
git clone https://github.com/your-username/Java-DBMS-Project.git
```

2. Open the project in NetBeans or your preferred Java IDE.

3. Configure the database connection in the source code.

4. Import the required database schema if provided.

5. Build and run the project.
